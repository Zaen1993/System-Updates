# RedAmon Module

RedAmon is a network reconnaissance and attack path planning tool. It scans target IPs for open ports, fingerprints services, and suggests exploitation strategies.

## Usage

```bash
python redamon.py <target_ip>
```

Configuration

Edit config.yaml to customize scan ports, threads, and AI model settings.

Dependencies

Install with:

```bash
pip install -r requirements.txt
```
